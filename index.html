Research Interests

I am broadly interested in design and implimentation aspects of all functional programming languges. My current focus is on the design and implimention of type systems.

Keywords: Functional programming, static type checkers, programming language semantics, type theory, logic.

Research Projects

  • Partial Type Constructors: Well-kinded types can be semantically bogus eg. Set (Int → Int). Can we teach Haskell (GHC) type checker to identify such bogus types, also along the way have a Monad instance for the Set datatype? [Code]
